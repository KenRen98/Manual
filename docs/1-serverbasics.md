# 操作入门

后台基本操作的简单教程，具体每个不同游戏的教程图文，请查看后续教程。

## 1.仪表盘

仪表盘区域会显示您所有的服务器，包括一些基本信息。正在安装中的服务器部分功能无法访问。  
**注：服务器安装/重装完成后会有邮件通知，无需一直刷新等待。**  
![服务器仪表盘](assets\1.0\header.png)  
<br>
