# 英灵神殿（Valheim）
一款灵感来源于维京文化并由程序生成的炼狱式1-10人探索生存游戏。

---

## 1. 推荐配置
**最少2G内存，最佳6G内存，最高8Gb内存**
**由于该游戏最多10人，所以能用的资源很有限**  
<br>
